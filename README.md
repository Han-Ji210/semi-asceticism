# Semi-asceticism (준고행) 
An interactive web map project documenting a 16.5km walk and the philosophical, sometimes pedantic, stream of consciousness that occurred along the way. Submitted as an assignment for a University Humanities class.

## Overview
This project visualizes the physical journey and the mental wandering of the author. By plotting a GPX track on a map and attaching essay points to specific coordinates, users can trace the exact route and read the thoughts—from academic anxiety and sensory awakening to existential questions about space, time, and human evolution—that popped up at each location.

## Features
* **Interactive GPX Tracking:** Displays the actual 16.5km route walked by the author using `leaflet-gpx`.
* **Dynamic Flight Animation:** The map automatically calculates the distance between story points and adjusts the flying speed and zoom level for a smooth presentation experience.
* **Story Popups:** Custom, scrollable popup windows containing photos and essays.
* **Sidebar Navigation:** An intuitive table of contents to easily navigate through the timeline of thoughts.

## Tech Stack
* HTML5 / CSS3
* Vanilla JavaScript
* [Leaflet.js](https://leafletjs.com/) (Interactive maps)
* [Leaflet-gpx](https://github.com/mpetazzoni/leaflet-gpx) (GPX file parsing)
